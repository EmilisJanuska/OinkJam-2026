Controls:
    Main Menu: ESCAPE
    Movement: WASD keys

    Temporary: Start / Stop combat with SPACE

Notes:
    - In the main menu, if you press new game it will reset the main scene back to the start

Known Bugs:

    so if you spam 'new game' in the main menu, it seems to send the player flying across the screen every second start.
